# Seekr
Seekr is a Lost &amp; Found University Portal, a web-based platform designed to modernize and streamline the traditional Lost and Found process within a university environment. The system provides a centralized platform where students can report lost items, submit found items, communicate with potential owners, and track recovery progress.
Features implemented include user registration and login, lost item posting, found item posting, categories, image upload support, search functionality, claim requests, comments, notifications, user profiles, admin dashboard, messaging system, status updates, moderation capabilities, and abuse reporting. Email notifications are planned for future implementation.

Conclusion
The Lost & Found University Portal provides a comprehensive solution for modernizing item recovery within a university environment. The platform integrates user management, item tracking, claims processing, communication tools, moderation features, and database-driven operations into a single scalable system. With future enhancements such as mobile applications, QR-code support, email notifications, and AI-assisted matching, the platform has strong potential for broader adoption beyond a single institution.
